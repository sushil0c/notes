Accessible Notepad

Instructions on how to use the Accessible Notepad:

  • Use the "Add Note" button to add a new note.
  • Sort notes by alphabet, creation date, or last edit date using the dropdown menu.
  • Search notes using the search bar for easy retrieval.
  • Submit feedback to help improve the app.
  • Export notes as a JSON file or import notes from a JSON file.
